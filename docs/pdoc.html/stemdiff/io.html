stemdiff.io

Input/output functions for package stemdiff.

Three types of stemdiff.io objects

  • Datafiles = files on disk, saved directly from a 2D-STEM detector.
  • Arrays = the datafiles converted to numpy array objects.
  • Images = the datafiles converted to PNG files.

General strategy for working with stemdiff.io objects

  • Datafiles and Images are usually not used directly, but just converted to Array objects.
  • All data manipulation (showing, scaling, saving ...) is done within Array objects.
  • Datafiles and Images have (intentionally) just a limited amount of methods, the most important of which is read - this method simply reads Datafile/Image to an array.

Additional stemdiff.io utilities

  • These are stored directly in the package, without any subclassing.
  • Mostly general routines for more convenient I/O within stemdiff package.

Examples how to use Datafiles, Arrays and Images

>>> # Show a datafile
>>> # (basic operation => there is Datafiles.function for it
>>> stemdiff.io.Datafiles.show(SDATA, filename)

>>> # Read a datafile to array
>>> # (basic operation => there is Datafiles.function for it
>>> arr = stemdiff.io.Datafiles.read(SDATA, filename)

>>> # Describe AND show the datafile
>>> # (more complex operation:
>>> # (1) read datafile to array - using Datafiles.read
>>> # (2) do what you need (here: describe, show) - using Arrays.functions
>>> arr = stemdiff.io.Datafiles.read(SDATA, datafile)
>>> stemdiff.io.Arrays.describe(arr, csquare=20)
>>> stemdiff.io.Arrays.show(arr, icut=1000, cmap='gray')

 89    Plot a few selected 2D diffraction patterns in a row one-by-one.
 90
 91    Parameters
 92    ----------
 93    data_to_plot : list of lists
 94        This object is a list of lists.
 95        The number of rows = the number of plotted diffractograms.
 96        Each row contains two elements:
 97        (i) data for diffractogram to plot and (ii) title of the plot.
 98        The data (first element of each row) can be:
 99        (i) PNG-file or (ii) 2D-array containing the 2D diffractogram. 
100    integer, optional, default is None
101        Cut of intensity;
102        if icut = 300, all image intensities > 300 will be equal to 300.
103    cmap : str - matplotlib.colormap name, optional, the default is 'viridis'
104        Matplotlib colormap for plotting of the diffractogram.
105        Other interesting or high-contrast options:
106        'gray', 'plasma', 'magma', ...
107        The full list of matplotlib colormaps:
108        `matplotlib.pyplot.colormaps()` 
109    output_file : str, optional, default is None
110        If this argument is given,
111        the plot is also saved in *output_file* image
112        with *dpi* resolution (dpi is specified by the following argument).
113    dpi : int, optional, default is 300
114        The (optional) argument gives resolution of (optional) output image. 
115
116    Returns
117    -------
118    None.
119    The output is the plot of the diffraction patterns on the screen.
120    If argument *ouput_file* is given, the plot is also saved as an image. 
121    '''
122    # Initialize
123    n = len(data_to_plot)
124    diffs = data_to_plot
125    fig,ax = plt.subplots(nrows=1, ncols=n)
126    # Plot 2D-diffractograms
127    for i in range(n):
128        # Read data to plot
129        data = diffs[i][0]
130        if type(data) == str:  # Datafile
131            if data.lower().endswith('.png'):  # ....PNG file, 2D-diffractogram
132                # we read image as '16bit'
133                # in this case, it works for 8bit images as well    
134                arr = Images.read(data, itype='16bit')
135            else:  # .......some other string not ending on PNG => unknown data
136                print('Unknown type of data to plot!')
137        elif type(data) == np.ndarray:  # Numpy array
138            if data.shape[0] == data.shape[1]:  # sqare array, 2D-diffractogram

140            else: # .....some other, non-square array => nonstandard array size

146        ax[i].set_title(my_title)

447

461        # (just simple removal of edges based on geometrical center!

464            Xsize,Ysize = arr.shape
465            xc,yc = (int(Xsize/2),int(Ysize/2))

473            plt.imshow(arr, cmap=cmap)
474            if colorbar:  # Add colorbar
475                plt.colorbar()
476            if center==True:  # Mark intensity center in the plot
477                xc,yc = Arrays.find_center(arr,csquare, cintensity)
478                plt.plot(yc,xc, 'r+', markersize=20)
479        # (b) Prepare 3D plot (option; if plt_type is not the default '2D')
480        else:  
481            if cmap==None:  # if cmap not selected, set default for 3D maps

486            Y = np.arange(Ysize)

488            # Create 3D-plot
489            from mpl_toolkits.mplot3d import Axes3D
490            fig = plt.figure()
491            ax = fig.add_subplot(111, projection='3d')
492            ax.plot_surface(
493                Xm,Ym,arr, cmap=cmap, linewidth=0, antialiased=False)
494            plt.tight_layout()
495        # (c) Show the plot
496        plt.show()

544    def find_center(arr, csquare=None, cintensity=None):
545        '''
546        Determine center of mass for 2D numpy array.
547        Array center = mass/intensity center ~ position of central spot.
548        Warning: In most cases, geometric center is NOT mass/intensity center.
549    
550        Parameters
551        ----------
552        arr : numpy 2D array
553            Numpy 2D array, whose center (of mass ~ intensity) we want to get.
554        csquare : integer, optional, default is None
555            Edge of a central square, from which the center will be determined.
556        cintensity : float in interval 0--1, optional, default is None
557            The intensity < maximum_intensity * cintensity is regarded as 0
558            (a simple temporary background removal in the central square).
559            
560        Returns
561        -------
562        xc,yc : integers
563            Coordinates of the intesity center = position of the primary beam.
564        '''
565        # Calculate center of array
566        if csquare:
567            # If csquare was given,
568            # calculate center only for the square in the center,
569            # in which we set background intensity = 0 to get correct results.
570            # a) Calculate array corresponding to central square
571            xsize,ysize = arr.shape
572            xborder = (xsize - csquare) // 2
573            yborder = (ysize - csquare) // 2
574            arr2 = arr[xborder:-xborder,yborder:-yborder].copy()
575            # b) Set intensity lower than maximum*coeff to 0 (background removal)
576            coeff = cintensity or 0.8
577            arr2 = np.where(arr2>np.max(arr2)*coeff, arr2, 0)
578            # c) Calculate center of intensity (and add borders at the end)
579            M = measure.moments(arr2,1)
580            (xc,yc) = (M[1,0]/M[0,0], M[0,1]/M[0,0])
581            (xc,yc) = (xc+xborder,yc+yborder)
582            (xc,yc) = np.round([xc,yc],2)
583        else:
584            # If csquare was not given,
585            # calculate center for the whole array.
586            # => Wrong position of central spot for non-centrosymmetric images!
587            M = measure.moments(arr,1)
588            (xc,yc) = (M[1,0]/M[0,0], M[0,1]/M[0,0])
589            (xc,yc) = np.round([xc,yc],2)
590        # Return final values            
591        return(xc,yc)

545    def find_center(arr, csquare=None, cintensity=None):
546        '''
547        Determine center of mass for 2D numpy array.
548        Array center = mass/intensity center ~ position of central spot.
549        Warning: In most cases, geometric center is NOT mass/intensity center.
550    
551        Parameters
552        ----------
553        arr : numpy 2D array
554            Numpy 2D array, whose center (of mass ~ intensity) we want to get.
555        csquare : integer, optional, default is None
556            Edge of a central square, from which the center will be determined.
557        cintensity : float in interval 0--1, optional, default is None
558            The intensity < maximum_intensity * cintensity is regarded as 0
559            (a simple temporary background removal in the central square).
560            
561        Returns
562        -------
563        xc,yc : integers
564            Coordinates of the intesity center = position of the primary beam.
565        '''
566        # Calculate center of array
567        if csquare:
568            # If csquare was given,
569            # calculate center only for the square in the center,
570            # in which we set background intensity = 0 to get correct results.
571            # a) Calculate array corresponding to central square
572            xsize,ysize = arr.shape
573            xborder = (xsize - csquare) // 2
574            yborder = (ysize - csquare) // 2
575            arr2 = arr[xborder:-xborder,yborder:-yborder].copy()
576            # b) Set intensity lower than maximum*coeff to 0 (background removal)
577            coeff = cintensity or 0.8
578            arr2 = np.where(arr2>np.max(arr2)*coeff, arr2, 0)
579            # c) Calculate center of intensity (and add borders at the end)
580            M = measure.moments(arr2,1)
581            (xc,yc) = (M[1,0]/M[0,0], M[0,1]/M[0,0])
582            (xc,yc) = (xc+xborder,yc+yborder)
583            (xc,yc) = np.round([xc,yc],2)
584        else:
585            # If csquare was not given,
586            # calculate center for the whole array.
587            # => Wrong position of central spot for non-centrosymmetric images!
588            M = measure.moments(arr,1)
589            (xc,yc) = (M[1,0]/M[0,0], M[0,1]/M[0,0])
590            (xc,yc) = np.round([xc,yc],2)
591        # Return final values            
592        return(xc,yc)

757        # Rescale

765        else:
766            arr = arr.astype('uint16')
767        # Return the modified array
768        return(arr)
